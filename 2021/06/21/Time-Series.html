Time Series

Jun 21, 2021 â€¢ 1 min read

Time Series is a type of data or a problem in machine learning. In Time Series, data points are generated by the time and follow some patterns such as trending or period. Time series is very popular problem in reality, especially in trading. The price of stock or number of product sales are time series examples.

Generate sample data

To generate sample data, we need to build a model for time series.

The simplest way to depict the time series is by using linear model:

$$y = x^T * \theta$$

Basically, it estimates a line between data points. And it try to optimize the residuals between estimated results and actual results. However, this way does not describe some essensials of time series problem, such as treding or period. So, we will use a more complex model to depict the time series.

$$y = intercept + trend(t) + period(t) + nuisance$$ 

import numpy as np 
m = 1000 
intercept = 50 
trend = 0.1
period = lambda t: np.sin(2*np.pi*t/180)
nuisance = np.random.normal(0,120, m)

t = np.arange(1000)
y = intercept + trend * t + 120*period(t) + nuisance[t]

import matplotlib.pyplot as plt 
plt.plot(y)

Stationary and difference

Time series has 2 important characteristics: Stationary and difference.

Stationary Time series is called stationary whenever its properties dont depend on time or they are unpreditable. In the figure above, there are only b and g are stationary.

Difference Difference is a way we calculate the difference between consecutive observations. That is the way we can make treding or seasonary data to be stationary.

$$d_1 = y_t - y_{t-1}$$ $$ = y_t - B^1y_t$$ $$ = y_t (1-B)$$

$$d_2 = (y_t - y_{t-1}) - (y_{t-1} - y_{t-2})$$ $$= (y_t - B^1y_1) - (B^1y_1 - B^2y_1)$$ $$=y_t(1-B)^2$$

so we have: $$d_n = y_t(1-B)^n$$
